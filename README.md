# projeto-dam
O AnunciosLoc é um sistema distribuído para gestão de anúncios baseados em localização geográfica. Permite que utilizadores publiquem mensagens (anúncios) associadas a locais específicos (coordenadas GPS ou redes WiFi). Quando outros utilizadores se aproximam dessas áreas, recebem os anúncios publicados, acumulando pontos de bónus.
